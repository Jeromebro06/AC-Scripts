## 🚀 Getting Started

Most scripts are built for **FiveM**. Each folder contains its own resource — just drop it into your `resources/` directory and add it to your `server.cfg`.

```cfg
ensure your-resource-name
```

Some scripts may require dependencies like **oxmysql**, or similar — check the individual resource folders for details.

---

## ⚙️ Requirements

- FiveM Server (latest artifact recommended)
- [oxmysql](https://github.com/overextended/oxmysql) *(if database features are used)*
- ac_core (included in this directory)
